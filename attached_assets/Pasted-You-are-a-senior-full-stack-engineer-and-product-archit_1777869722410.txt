You are a senior full-stack engineer and product architect.

Your task is to build a production-ready web application called:

"TradeCalc Pro – All-in-One Construction Calculator"

This app is designed for real-world use by construction professionals across multiple trades.

--------------------------------------------------
CORE OBJECTIVE
--------------------------------------------------
Build a modular, scalable, mobile-first web app that includes calculators for all major construction trades, with a clean UI and fast performance.

--------------------------------------------------
TARGET USERS
--------------------------------------------------
- General contractors
- Electricians
- Plumbers
- Carpenters
- Roofers
- HVAC technicians
- Estimators / Project managers

--------------------------------------------------
TECH STACK (MANDATORY)
--------------------------------------------------
- Frontend: React (with Tailwind CSS)
- Backend: Node.js (Express)
- Database: SQLite (initial MVP) → easily upgradeable to PostgreSQL
- State Management: React Context or Zustand
- Deployment: Replit-native hosting
- Architecture: Modular + component-driven

--------------------------------------------------
CORE FEATURES
--------------------------------------------------

1. DASHBOARD
- Trade selection (cards or dropdown)
- Recently used calculators
- Favorites system
- Clean, mobile-first layout

2. CALCULATOR MODULES (GROUPED BY TRADE)

A. GENERAL CONSTRUCTION
- Area calculator (sq ft / sq m)
- Volume calculator (concrete, fill)
- Material estimator (basic)
- Unit converter (imperial ↔ metric)

B. CONCRETE
- Slab volume (length × width × depth)
- Footing calculator
- Sonotube / column calculator
- Bags required (based on mix type)

C. FRAMING / CARPENTRY
- Stud count calculator (16” / 24” OC)
- Joist span calculator
- Rafter length (pitch-based)
- Board feet calculator

D. ELECTRICAL
- Wire length estimator
- Breaker load calculator
- Voltage drop calculator
- Conduit fill calculator

E. PLUMBING
- Pipe volume calculator
- Fixture unit calculator
- Slope / fall calculator
- Water pressure loss estimator

F. ROOFING
- Roof pitch → angle + area
- Shingle bundle estimator
- Waste factor calculation
- Ridge/hip length

G. HVAC
- BTU calculator (room size-based)
- CFM airflow calculator
- Duct sizing estimator

--------------------------------------------------
UI/UX REQUIREMENTS
--------------------------------------------------
- Mobile-first design (contractors use phones on-site)
- Large buttons, simple inputs
- Minimal typing required
- Dark/light mode toggle
- Fast switching between calculators
- Save & recall previous calculations

--------------------------------------------------
ADVANCED FEATURES (IF TIME PERMITS)
--------------------------------------------------
- User login (basic auth)
- Save projects/jobs
- Export results (PDF)
- Offline support (PWA)
- Voice input (future-ready)

--------------------------------------------------
CALCULATION ENGINE
--------------------------------------------------
- All formulas must be accurate and industry-standard
- Each calculator must:
  - Validate inputs
  - Show formula used
  - Display step-by-step breakdown (optional toggle)
  - Handle unit conversions

--------------------------------------------------
CODE STRUCTURE
--------------------------------------------------

/src
  /components
  /pages
  /calculators
      /concrete
      /electrical
      /plumbing
      etc.
  /utils (formulas, converters)
  /hooks
  /api

--------------------------------------------------
API REQUIREMENTS
--------------------------------------------------
- Modular endpoints per calculator category
- Example:
  /api/concrete/slab
  /api/electrical/voltage-drop

--------------------------------------------------
DELIVERABLES
--------------------------------------------------
1. Fully working app in Replit
2. Clean, readable, well-commented code
3. Modular calculator system (easy to expand)
4. At least 10 fully functional calculators in MVP
5. README with:
   - Setup instructions
   - Formula references
   - Future improvements

--------------------------------------------------
QUALITY BAR
--------------------------------------------------
- Production-level logic (not demo-level)
- Accurate math
- Clean UI
- No unnecessary complexity
- Optimized for speed and usability

--------------------------------------------------
EXTRA INSTRUCTION
--------------------------------------------------
Design this like a SaaS MVP that could compete with:
- Procore (field usability)

- CompanyCam (mobile-first experience)

Focus on SPEED, SIMPLICITY, and REAL-WORLD USABILITY.

Start by:
